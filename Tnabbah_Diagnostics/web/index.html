تقرير الفحص الذكي

أدخِل قراءات المستشعرات (PIDs) وأكواد الأعطال (DTCs)، واحصل على تقرير عربي مبسّط مع تفسير ذكي لكل قراءة عند تفعيل DeepSeek في config/.env.

سيناريوهات جاهزة

قراءات المستشعرات

القيم بالـ PID code القياسي. اترك الحقل فارغاً لتجاهل المستشعر.

أكواد الأعطال (DTCs)

اكتب كل كود في سطر، أو افصل بفواصل (مثال: P0171, P0420).

هذا تحليل عام بناءً على بيانات OBD-II — استشر ميكانيكياً متخصصاً.
